Deals & Cashflows Application created by Python Django
This is a website contains two tables to implement CRUD data operations.
Demo link: http://jean.athenagu.com/admin
Demo link2: http://jean.athenagu.com

Running method:
1) Install Python
2) Install Django
3) command line go to the file folder: cd dealscashflows
3) command line: python manage.py runserver
4) Browser: http://127.0.0.1:8000/admin  (username&password: admin)to see the admin user interface
5) Browser: http://127.0.0.1:8000 to see the basic page (which uses a different method to implement)
